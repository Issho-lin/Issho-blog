# дёҖгҖҒдәӢд»¶еҫӘзҺҜжҳҜд»Җд№Ҳ

еҗ„з§ҚжөҸи§ҲеҷЁдәӢд»¶еҗҢж—¶и§ҰеҸ‘ж—¶пјҢиӮҜе®ҡжңүвјҖдёӘе…ҲжқҘеҗҺеҲ°зҡ„жҺ’йҳҹй—®йўҳгҖӮеҶіе®ҡиҝҷдәӣдәӢд»¶еҰӮдҪ•жҺ’йҳҹи§ҰеҸ‘зҡ„жңәеҲ¶пјҢе°ұжҳҜдәӢд»¶еҫӘзҺҜгҖӮ иҝҷдёӘжҺ’йҳҹвҫҸдёәд»Ҙ JavaScript ејҖеҸ‘иҖ…зҡ„в»ҶеәҰжқҘзңӢпјҢдё»иҰҒжҳҜеҲҶжҲҗдёӨдёӘйҳҹеҲ—пјҡ

  • вјҖдёӘжҳҜ JavaScript еӨ–йғЁзҡ„йҳҹеҲ—гҖӮеӨ–йғЁзҡ„йҳҹеҲ—дё»иҰҒжҳҜжөҸи§ҲеҷЁеҚҸи°ғзҡ„еҗ„зұ»дәӢд»¶зҡ„йҳҹеҲ—пјҢж ҮеҮҶвҪӮд»¶дёӯз§°д№ӢдёәTask QueueгҖӮдёӢвҪӮдёӯдёәдәҶвҪ…дҫҝзҗҶи§Јз»ҹвјҖз§°дёәеӨ–йғЁйҳҹеҲ—гҖӮ
  • еҸҰвјҖдёӘжҳҜ JavaScript еҶ…йғЁзҡ„йҳҹеҲ—гҖӮиҝҷйғЁеҲҶдё»иҰҒжҳҜ JavaScript еҶ…йғЁжү§вҫҸзҡ„д»»еҠЎйҳҹеҲ—пјҢж ҮеҮҶдёӯз§°д№ӢдёәMicrotask QueueгҖӮдёӢвҪӮдёӯдёәдәҶвҪ…дҫҝзҗҶи§Јз»ҹвјҖз§°дёәеҶ…йғЁйҳҹеҲ—гҖӮ еҖјеҫ—жіЁж„Ҹзҡ„жҳҜпјҢиҷҪз„¶дёәдәҶеҘҪзҗҶи§ЈжҲ‘д»¬з®ЎиҝҷдёӘеҸ«йҳҹеҲ— (Queue)пјҢдҪҶжҳҜжң¬иҙЁдёҠжҳҜжңүеәҸйӣҶеҗҲ (Set)пјҢеӣ дёәдј з»ҹзҡ„йҳҹеҲ—йғҪжҳҜе…Ҳиҝӣе…ҲеҮәпјҲFIFOпјүзҡ„пјҢвҪҪиҝҷвҫҘзҡ„йҳҹеҲ—еҲҷдёҚз„¶пјҢжҺ’еҲ°жңҖеүҚвҫҜдҪҶжҳҜжІЎжңүж»ЎвҫңжқЎд»¶д№ҹжҳҜдёҚдјҡжү§иЎҢзҡ„пјҲвҪҗеҰӮеӨ–йғЁйҳҹеҲ—вҫҘеҸӘжңүвјҖдёӘ setTimeout зҡ„е®ҡж—¶д»»еҠЎпјҢдҪҶжҳҜж—¶й—ҙиҝҳжІЎжңүеҲ°пјҢжІЎжңүж»ЎвҫңжқЎд»¶д№ҹдёҚдјҡжҠҠд»–еҮәеҲ—жқҘжү§вҫҸпјүгҖӮ

# еӨ–йғЁйҳҹеҲ—

еӨ–йғЁйҳҹеҲ—пјҲTask QueueпјүпјҢйЎҫеҗҚжҖқд№үе°ұжҳҜ JavaScript еӨ–йғЁзҡ„дәӢд»¶зҡ„йҳҹеҲ—пјҢиҝҷвҫҘжҲ‘д»¬еҸҜд»Ҙе…ҲеҲ—дёҫвјҖдёӢжөҸи§ҲеҷЁдёӯиҝҷдәӣеӨ–йғЁдәӢд»¶жәҗпјҲTaskSourceпјүпјҢд»–д»¬дё»иҰҒжңүпјҡ

  • DOM ж“ҚдҪң (в»ҡвҫҜжёІжҹ“)
  • вҪӨжҲ·дәӨдә’ (вҝҸж ҮгҖҒй”®зӣҳ)
  • вҪ№з»ңиҜ·жұӮ (Ajax зӯү)
  • History API ж“ҚдҪң
  • е®ҡж—¶еҷЁ (setTimeout зӯү) еҸҜд»Ҙи§ӮеҜҹеҲ°пјҢиҝҷдәӣеӨ–йғЁзҡ„дәӢд»¶жәҗеҸҜиғҪеҫҲеӨҡпјҢдёәдәҶвҪ…дҫҝжөҸи§ҲеҷЁвјҡе•ҶдјҳеҢ–пјҢHTML ж ҮеҮҶдёӯжҳҺзЎ®жҢҮеҮәвјҖдёӘдәӢд»¶еҫӘзҺҜз”ұвјҖдёӘжҲ–еӨҡдёӘеӨ–йғЁйҳҹеҲ—пјҢвҪҪжҜҸвјҖдёӘеӨ–йғЁдәӢд»¶жәҗйғҪжңүвјҖдёӘеҜ№еә”зҡ„еӨ–йғЁйҳҹеҲ—гҖӮдёҚеҗҢдәӢд»¶жәҗзҡ„йҳҹеҲ—еҸҜд»ҘжңүдёҚеҗҢзҡ„дјҳе…Ҳзә§пјҲдҫӢеҰӮеңЁвҪ№з»ңдәӢд»¶е’ҢвҪӨжҲ·дәӨдә’д№Ӣй—ҙпјҢжөҸи§ҲеҷЁеҸҜд»Ҙдјҳе…ҲеӨ„зҗҶвҝҸж ҮвҫҸдёәпјҢд»ҺвҪҪи®©вҪӨжҲ·ж„ҹи§үжӣҙеҠ жөҒзЁӢпјүгҖӮ

# еҶ…йғЁйҳҹеҲ—

еҶ…йғЁйҳҹеҲ—пјҲMicrotask QueueпјүпјҢеҚі JavaScript иҜӯвҫ”еҶ…йғЁзҡ„дәӢд»¶йҳҹеҲ—пјҢеңЁ HTML ж ҮеҮҶдёӯпјҢе№¶жІЎжңүжҳҺзЎ®и§„е®ҡиҝҷдёӘйҳҹеҲ—зҡ„дәӢд»¶жәҗпјҢйҖҡеёёи®Өдёәжңүд»ҘдёӢвјҸз§Қпјҡ

  • Promiseзҡ„жҲҗеҠҹ (.then) дёҺеӨұиҙҘ (.catch)
  • MutationObserver
  • Object.observe (е·Іеәҹејғ)
